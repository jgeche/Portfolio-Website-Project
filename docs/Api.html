What is the project?

The eight project of my Front-End Web Development journey was the API Project. The project involved accessing information from an API and displaying it on a webpage. Twelve employees must display on the page from the API using Fetch and every time the page is refreshed another twelve employees are displayed. Employee information must match the mockups and a modal window must be added to the picture with the correct information.

How was it put together?

The first task of the API project was to create the layout of the webpage through HTML and CSS. Next I added the Javascript involved in creating the project. Necessary variables including the API was added first, followed by the Fetch syntax. Then, I created a function that properly displayed the necessary information to the HTML document. Next I added a modal to the HTML and Javascript documents. Finally, I made sure the project was functioning properly before submitting the project.
