What is it?

The sixth project in the Treehouse TechDegree program was the Game Show App, a game made of Javascript where you would guess a phrase using letters on the keyboard. Each wrong letter loses you a heart and five wrong guesses means you lose the game. There is an overlay display page that starts the page that must be shown first, the game must work properly with Javascript using five different arrays for the answer and must show a winner and loser page to the user. The various overlays for the game were provided as the project was mainly Javascript involved.

How was it put together?

As this was my first Javascript project I did a lot of studying through Treehouse to make sure I had the right Javascript syntax to make it work properly. I first created the variables needed for creating the game show app within the HTML document. Next, I created an Event Listener to remove the start page overlay when the start game button is clicked. I then created an array of five phrases and returned a random phrase to the display. Next, I created a function to see if the clicked letter is wrong or correct. I then created an Event Listener for mouse clicks when the game is playing, reducing a heart if the answer is incorrect. Finally I created a function that showed the winner or loser display if user chose the right answer or ran out of hearts on the app.

What were the challenges?

The first time I got my project from Treehouse I had one small typo that I had to adjust so it works correctly. In the AddtoPhrase function I needed to have a space within the quotation marks and the app will work correctly. I fixed the small issue and passed the project the second time around. The biggest challenge was learning the correct syntax for Javascript functions and Event Listeners and making sure they work correctly. After some trial and error I got them all working.

What did I learn?

Javascript is used all over the web including social media, major websites and interactivity of all kinds. It was a very vital experience for me to create a game made mostly of Javascript as Javascript is very much a part of the daily life of a web developer. Creating the correct syntax for Javascript will be something that I will continually work on in the future to make any website I make the most interactive it can be.
