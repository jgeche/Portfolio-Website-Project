What is it?

The seventh project of my Treehouse TechDegree journey was the WebApp Dashboard project. The WebApp Dashboard is an interactive app website with SVGs, social media notifications and posts, three charts imported through a plugin and a messaging service used to message other users. The three charts for the webpage are to be imported through a Javascript library.

How was it put together?

The first task in the WebApp project was to create the layout that matches the mockups given by Treehouse in mobile, tablet and desktop views. I then styled and changed the SVGs to correctly fit the mockups. Next was adding the Javascript to the notification button, alert section and and importing the three graphs. Finally, I added Javascript to the message section so an alert would display if not all data was filled out.

What were the challenges?

I submitted my project through Treehouse and there were a couple errors that I needed to fix. The size of the line chart was surpassing the margins of the page and a greater than symbol is missing which did not match the mockup. Once I fixed those the app worked as it should and I passed the project the second time around.

What did I learn?

Being able to use Javascript, HTML and CSS cohesively is important to any well made website. The WebApp Dashboard project was a large project and allowed me to use all my skills involved in creating a large website. It took some trial and error but eventually I was able to get the project work correctly.
